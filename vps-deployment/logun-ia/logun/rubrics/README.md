# Rubrics

Rubricas de pontuação por desafio: definem como uma resposta é convertida em nota.

São mantidas fora do repositório público por serem, na prática, parte do gabarito.
Expor as rubricas permitiria otimizar respostas para o critério em vez de para o
problema. Apenas a estrutura de pastas é versionada; as rubricas ficam no ambiente
de execução da VPS e são carregadas por `logun/core/rubric_loader.py`.
