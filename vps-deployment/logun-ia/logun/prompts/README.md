# Prompts

Modelos de prompt que a logun-ia usa para instruir os provedores de avaliação.

Os arquivos de prompt não fazem parte do repositório público. Eles determinam
como o serviço orienta os modelos de linguagem a julgar uma resposta, e expô-los
facilitaria contornar ou manipular a avaliação. Apenas a estrutura de pastas é
versionada; os prompts ficam no ambiente de execução da VPS.
