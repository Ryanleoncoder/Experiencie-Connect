# Logun contexts

Contextos de avaliação dos desafios assistidos por IA: um arquivo por desafio,
com o cenário, os critérios e as referências usadas para pontuar respostas.

O conteúdo é mantido fora do repositório público de forma intencional. Esses
arquivos funcionam como gabarito — descrevem o que caracteriza uma boa resposta
e como a nota é formada. Publicá-los permitiria ajustar as respostas ao critério
em vez de resolver o desafio.

A estrutura é versionada por meio deste README; os contextos existem apenas no
ambiente de execução.
